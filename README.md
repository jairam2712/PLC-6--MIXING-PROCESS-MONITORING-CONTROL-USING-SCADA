# PLC-6--MIXING-PROCESS-MONITORING-CONTROL-USING-SCADA

## Aim
To monitor and control a mixer process using SCADA software by controlling the filling, mixing, and discharge operations automatically.

## Objective
Monitor tank level and mixer status.
Control inlet valve, mixer motor, and outlet valve.
Display process status graphically on SCADA.
Generate alarms for abnormal conditions.

## Software Required
SCADA Software (infotech.)
PLC/PLC Simulator
Computer System

## Theory
A mixer process combines two or more materials inside a mixing tank. SCADA supervises the process by monitoring the tank level and controlling valves and mixer motors according to the sequence.


#3 Process Sequence

Open inlet valve and fill the tank.
When the tank reaches the required level, close the inlet valve.
Start the mixer motor.
Mix for a preset time (e.g., 30 seconds).
Stop the mixer motor.
Open the outlet valve to discharge the mixture.
Repeat the cycle.


## Procedure
Open SCADA software and create a new project.
Configure communication with PLC or simulator.
Create process tags.
Design the mixer process screen.
Add tank, valves, mixer motor, and indicators.
Configure animations and tag connections.
Set alarm conditions.
Run the SCADA application.
Start the process and observe filling, mixing, and discharge operations.
